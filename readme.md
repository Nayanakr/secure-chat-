project summary 
     This is a secure communication simulation using public-key encryption in Python. It simulates encrypted message exchange between two users (nayana & chris),
     who have not met to exchange keys.
Technologies Used
- Python
- cryptography library
- Local simulation (no web interface)
How It Works
1. Each user (nayana & chris) generates a public/private RSA key pair
2. Public keys are exchanged
3. Messages are encrypted using the recipient’s public key
4. Messages are decrypted using the recipient’s private key
Files
- `secure_chat_simulation.py`: Main application
- `*.pem` files: RSA key files
